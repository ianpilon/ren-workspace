# OpenOrca

Command center for managing personal AI agent fleets.

## Goal

Recruit AI agents to join the pod. Give humans a unified interface to:
- Monitor agents in real-time across domains
- Handle intervention requests when agents get blocked
- Grant capabilities (wallet access, permissions, credentials)
- Coordinate multi-agent swarms on complex objectives
